## Overview

DefleMask is a free (donationware), closed-source, cross-platform multi-system
chiptune tracker by Leonardo "Delek" Demartino, first released ~2011-2013. It
targets many soundchips; the Commodore 64 SID is one of them, and on export it
generates a C64 replay embedded in the resulting `.sid` (Player-ID tags these
files "DefleMask"). It is included here because it is a genuinely **public,
widely-used tool** — but note it is *not* open source, which changes what a
card can honestly say about it.

## Quirks & gotchas

See the `quirks` array. The load-bearing point: **public ≠ open-source**.
DefleMask is freely available but its source is closed, so — unlike
[goattracker](goattracker.md)/[cheesecutter](cheesecutter.md)/[sidwizard](sidwizard.md)
— there is no source or disassembly to verify its runtime against. The
open-source **Furnace** tracker reads DefleMask `.dmf` modules and is the
practical reference for the *module* format; the *C64 replay* it emits remains
undocumented. Early C64 exports were also criticized as memory-heavy.

## Disassembly notes

None. The C64 export replay is code-generated by a closed-source tool and has
not been disassembled here. A future pass could disassemble a representative
DefleMask-tagged `.sid` (init/play from its PSID header) and trace it through
`sidm2-siddump` the way the native players were handled — that would be the
only route to real memory/format facts, since no source is available.

## Verification

**Not verified — `status: stub`, and unusually gap-heavy on purpose.** Only
identity facts (author, year, platform, closed-source status) are confirmed,
from deflemask.com, the CSDb release, and the cached SIDId entry. Every
runtime field is `TODO` because the export engine is closed — an honest stub
beats a guessed memory map.
